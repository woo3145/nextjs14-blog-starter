## 학습

### DB

- [MySQL for Developers (PlanetScale)](https://planetscale.com/learn/courses/mysql-for-developers/introduction/course-introduction) - 강사님이 복잡한 문장이나 어휘를 사용하지 않아 굉장히 편한 영어로 스트레스 안받음(영자막 o)
- [Database Scaling (PlanetScale)](https://planetscale.com/learn/courses/database-scaling/scaling/path-to-scaling?autoplay=1) - 말도 빠르고 문장도 길어서 다시 외국인과의 벽을 느끼지만 DB의 점진적인 확장을 다루는 흥미로운 주제(영자막 o)
